MusicGateAway

A lightweight TIDAL proxy with a built-in web UI. Search, stream, and download music in lossless quality.

Download GitHub

What it does

🎧

Lossless Streaming

Stream FLAC (16-bit/44.1kHz) or high-quality AAC directly in your browser.

🔍

Search Everything

Find tracks, albums, and artists from TIDAL's full catalog with instant results.

📥

Download & Tag

Download tracks with full metadata: title, artist, album, track number, and cover art.

🎛

Built-in Web UI

No separate frontend setup. The UI is embedded in the single binary and served automatically.

⚡

Single Binary

Written in Rust with Axum. One executable, no runtime dependencies. Just run it.

🔌

REST API

Full JSON API for search, metadata, streaming, and downloads. Build your own tools on top.
